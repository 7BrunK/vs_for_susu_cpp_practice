**Условия:**
1. Дан массив из N элементов. Вводится число K – количество блоков (K<<N).
Упорядочить элементы по убыванию в каждом блоке.
2. Написать программу, которая проверяет, является ли введенная с клавиатуры
строка вещественным числом.

  Исходный массив: 32 80 39 23 42 82 18 6 41 85

  Размеры блоков: 2 2 2 2 2
  Массив после сортировки блоков: 80 32 39 23 82 42 18 6 85 41
