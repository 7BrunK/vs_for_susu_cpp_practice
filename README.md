# Kormilin Alexander ET-113, var - 13

## Условия задач

1.  Дан массив из N элементов. Вводится число K – количество блоков (K << N). Упорядорить элементы по убыванию в каждом блоке.
2.  Написать программу, которая проверяет, является ли введенная с клавиатуры строка вещественным числом.

---

## Тесты

### 1. Сортировка блоков в массиве (МАССИВЫ)

-   **Тест 1 (Блоки по 2 элемента)**
    -   Введите размер массива N: `10`
    -   Исходный массив: `32 80 39 23 42 82 18 6 41 85`
    -   Введите количество блоков K: `5`
    -   Размеры блоков: `2 2 2 2 2`
    -   Массив после сортировки блоков: `80 32 39 23 82 42 18 6 85 41`

-   **Тест 2 (Один блок)**
    -   Введите размер массива N: `10`
    -   Исходный массив: `56 35 49 25 79 25 19 73 21 35`
    -   Введите количество блоков K: `1`
    -   Размеры блоков: `10`
    -   Массив после сортировки блоков: `79 73 56 49 35 35 25 25 21 19`

-   **Тест 3 (Блоки разного размера)**
    -   Введите размер массива N: `10`
    -   Исходный массив: `18 47 88 76 48 97 20 91 29 25`
    -   Введите количество блоков K: `8`
    -   Размеры блоков: `2 2 1 1 1 1 1 1`
    -   Массив после сортировки блоков: `47 18 88 76 48 97 20 91 29 25`

#### Исключительные ситуации (Задача 1)

-   **Некорректный размер массива**
    -   Введите размер массива N: `0`
    -   Ошибка: `N <= 0`

-   **Некорректное количество блоков (слишком большое)**
    -   Введите размер массива N: `10`
    -   Исходный массив: `91 56 17 80 65 38 40 85 7 58`
    -   Введите количество блоков K: `200`
    -   Ошибка: `Некорректное значение K!`

-   **Некорректное количество блоков (отрицательное)**
    -   Введите размер массива N: `10`
    -   Исходный массив: `53 0 57 31 34 10 40 3 16 17`
    -   Введите количество блоков K: `-2`
    -   Ошибка: `Некорректное значение K!`

### 2. Проверка строки на вещественное число (СТРОКИ)

-   **Тест 1 (Целое число)**
    -   Введите строку для проверки: `100`
    -   Строка `100` является вещественным числом.

-   **Тест 2 (Положительная дробь)**
    -   Введите строку для проверки: `0.001`
    -   Строка `0.001` является вещественным числом.

-   **Тест 3 (Отрицательное число)**
    -   Введите строку для проверки: `-20.01`
    -   Строка `-20.01` является вещественным числом.

-   **Тест 4 (Лишний символ в конце)**
    -   Введите строку для проверки: `20.02+`
    -   Строка `20.02+` НЕ является вещественным числом.

-   **Тест 5 (Две точки)**
    -   Введите строку для проверки: `0.01.1`
    -   Строка `0.01.1` НЕ является вещественным числом.

-   **Тест 6 (Буквенная строка)**
    -   Введите строку для проверки: `string`
    -   Строка `string` НЕ является вещественным числом.

-   **Тест 7 (Смешанный формат)**
    -   Введите строку для проверки: `-2st.01`
    -   Строка `-2st.01` НЕ является вещественным числом.

#### Исключительные ситуации (Задача 2)

-   **Пустая строка**
    -   Введите строку для проверки:
    -   Ошибка: `Введена пустая строка!`
